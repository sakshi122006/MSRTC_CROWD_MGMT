# MSRTC_CROWD_MGMT
Predicts how crowded an MSRTC bus will be on a given route, day, and time slot.   Example: User selects “Solapur → Pune, Monday, 8:00 AM” → Model says “High crowd – 85% seats likely full. Better take 8:30 AM.”
The MSRTC Crowd Management System is a predictive application designed to estimate how crowded a bus will be based on route, day, and time. It helps passengers choose better travel timings and reduces inconvenience caused by overcrowding.
